# Phishing Detection & Awareness System

## Overview
This project focuses on identifying phishing emails and educating users on how to detect and prevent phishing attacks.

## Features
- Detection of phishing indicators
- Email classification (Phishing / Safe)
- Real-world phishing example analysis
- User awareness and prevention guidelines

## Tools Used
- Email analysis techniques
- Microsoft Word (Report)
- Browser inspection tools

## Key Learnings
- Identifying spoofed domains
- Detecting malicious intent in emails
- Risk communication
